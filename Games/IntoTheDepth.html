Into The Depth

Description

Since I love Hack and Slay games like Diablo or Path of Exile, I wanted to create one myself. This project is the biggest solo project I've worked on so far and it's far from finished. It was primarily for learning and I never intended to release it. The primary goal is to get deeper into the dungeon and defeat the final boss. There are 10 depth levels and to progress deeper, you have to complete the levels before. If you die, you lose all the new found loot. So the core game loop consists of leveling up, find better gear and getting stronger.

Main Features

  • procedural generated dungeons (from my PDG-Project)

  • loot system with different rarities and stat modifiers

  • character stats system like armor, critical hit damage or magical damage

  • ability system
