Procedural Dungeon Generation

Description

The generator is written in C# and can be used in Unity. It procedurally generates dungeon-like structures for top down based games like Diablo at runtime. For dungeons with less than 300 rooms, the generation is done in under a second. The generator provides a wide range of different parameters to manipulate the structure and shape of the resulting dungeon, giving the game designer control over the outcome. The Algorithm works on a grid, so predefined asset tiles can be used to generate floors and walls.
